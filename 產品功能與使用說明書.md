## 🎯 產品概述

### 什麼是 ReelMind？

**ReelMind** 是一款專為短影音創作者設計的 AI 智能體平台，透過人工智慧技術協助您：

- 🎬 **快速生成短影音腳本**：30 秒內產出專業腳本
- 🎭 **建立個人品牌定位**：AI 協助分析並建立您的 IP 人設
- 📊 **規劃內容策略**：14 天短影音內容規劃，不再為選題煩惱
- 💡 **提供選題靈感**：根據您的帳號定位推薦適合的主題

### 支援平台

- ✅ **Instagram Reels**
- ✅ **YouTube Shorts**
- ✅ **TikTok**
- ✅ **其他短影音平台**

## 🚀 核心功能介紹

### 功能一：一鍵生成模式（Mode3）

**適合對象**：追求效率、需要快速產出腳本的創作者

#### 主要功能

1. **帳號定位生成**
   - AI 分析您的背景和目標，生成個人品牌定位
   - 包含目標受眾、傳達目標、內容方向等完整分析

2. **選題推薦**
   - 根據帳號定位，AI 推薦適合的短影音主題
   - 提供多個選題方向，讓您有更多選擇

3. **短影音腳本生成**
   - 生成完整的 30 秒短影音腳本
   - 包含以下元素：
     - 📝 主題設定
     - 🎣 開場鉤子（Hook）
     - 💎 核心內容
     - 📢 行動呼籲（CTA）
     - 🎬 畫面描述
     - 📱 發佈文案

4. **腳本結構選擇**
   - 支援 5 種不同的腳本結構（A/B/C/D/E）
   - 可根據內容類型選擇最適合的結構

5. **階段性驗證機制**
   - 確保每個階段完成後再進行下一步
   - 避免跳過重要步驟

#### 使用流程

```
1. 選擇平台（Instagram / YouTube Shorts / TikTok）
2. 輸入主題或關鍵字
3. 點擊「生成帳號定位」→ 查看並確認
4. 點擊「生成選題推薦」→ 選擇喜歡的選題
5. 選擇腳本結構（A/B/C/D/E）
6. 點擊「生成短影音腳本」→ 完成！
7. 儲存到創作者資料庫或匯出為 CSV
```

### 功能二：IP 人設規劃模式（Mode1）

**適合對象**：想要深度建立個人品牌、需要長期規劃的創作者

#### 主要功能

1. **深度對話建立個人品牌檔案**
   - 透過與 AI 深度對話，記錄您的人生曲線和個人特色
   - AI 會記住您之前說過的內容，生成結果越來越精準

2. **帳號定位生成**
   - AI 協助建立完整的 IP 人設，包含：
     - 🎯 目標受眾分析
     - 📢 傳達目標
     - 🎭 帳號定位
     - 📝 內容方向
     - 🎨 風格調性
     - ⚡ 差異化優勢

3. **選題方向（影片類型配比）**
   - 根據帳號定位生成適合的影片類型、佔比、目的和內容方向建議
   - 以表格形式呈現，一目了然

4. **短影音腳本生成**
   - 生成完整的短影音腳本
   - 包含台詞內容、畫面描述、資訊融入建議、字幕建議、音效建議、發佈文案等

5. **14 天規劃**
   - 快速生成 14 天短影音內容規劃
   - 每天的主題和方向都已規劃好

6. **今日腳本**
   - 快速生成今日的短影音腳本
   - 適合需要立即產出內容的創作者

7. **過往紀錄管理**
   - 查看、選擇、匯出或刪除所有歷史生成記錄
   - 使用標籤式導航（帳號定位、選題方向、短影音腳本）
   - 支援標題編輯功能
   - 快速查看和展開完整內容

8. **自由搭配設定**
   - 可以從過往紀錄中選擇不同類型的設定（帳號定位、選題方向、短影音腳本）
   - 自由組合後與 AI 討論，創造更多可能性

9. **重新定位功能**
   - 點擊「重新定位」快速按鈕
   - AI 會顯示「短影音內容策略矩陣」表格
   - 協助重新規劃帳號定位

#### 使用流程

```
1. 綁定 LLM API 金鑰（必須）
2. 與 AI 對話，建立個人品牌檔案
3. 點擊「IP Profile」按鈕 → 生成帳號定位
4. 點擊「14天規劃」按鈕 → 生成選題方向
5. 點擊「今日腳本」按鈕 → 生成短影音腳本
6. 在 AI 回應中點擊「儲存」按鈕 → 儲存到生成結果
7. 在生成結果彈跳視窗中查看、選擇、編輯或刪除記錄
8. 可以選擇過往的設定，自由組合後與 AI 討論
```

#### 重要提醒

- ⚠️ **必須綁定 LLM API 金鑰**：Mode1 必須綁定自己的 LLM API 金鑰才能與 AI 對談
- 🔑 **支援的 LLM**：Gemini 和 OpenAI
- 📖 **如何取得金鑰**：點擊「如何取得」按鈕，直接導向實戰指南教學文章

### 功能三：創作者資料庫（UserDB）

**功能說明**：統一管理您的所有創作內容和個人資料

#### 主要功能

1. **個人資料管理**
   - 查看和編輯個人資訊
   - 查看當前訂閱狀態、到期時間

2. **我的腳本管理**
   - 查看所有已生成的腳本
   - 查看完整腳本內容
   - 下載 PDF 或 CSV 檔案
   - 刪除不需要的腳本

3. **帳號定位記錄**
   - 查看所有已保存的帳號定位記錄
   - 支援展開查看完整內容

4. **選題記錄**
   - 查看所有已保存的選題記錄
   - 支援展開查看完整內容

5. **IP 人設規劃結果**
   - 查看所有 IP 人設規劃的生成結果
   - 分類顯示：帳號定位、選題方向、短影音腳本
   - 支援展開查看完整內容

6. **訂閱狀態管理**
   - 查看當前訂閱狀態、到期時間
   - 取消自動續費功能
   - 查看訂單記錄

7. **使用統計**
   - 查看使用情況統計
   - 了解自己的使用習慣

8. **AI API Key 金鑰管理**
   - 綁定自己的 LLM API 金鑰（支援 Gemini 和 OpenAI）
   - 測試 API 金鑰是否有效
   - 選擇使用的模型
   - 系統會優先使用用戶提供的金鑰

9. **資料匯出**
   - 可將所有資料匯出為 Excel 檔案
   - 方便備份和管理

## ❓ 常見問題

### Q1: Mode1 和 Mode3 有什麼差別？

**A:** 
- **Mode3（一鍵生成）**：快速、結構化，適合效率導向的用戶，不需要深度對話
- **Mode1（IP 人設規劃）**：深度、互動性強，適合需要建立完整個人品牌的用戶，AI 會記住您的對話歷史

### Q2: 為什麼 Mode1 需要綁定 LLM API 金鑰？

**A:** Mode1 使用深度對話功能，需要大量的 AI 計算資源。為了提供更好的服務品質和穩定性，我們要求用戶綁定自己的 LLM API 金鑰。這樣可以：
- 確保服務穩定性
- 提供更好的對話體驗
- 避免資源限制

### Q3: 如何取得 LLM API 金鑰？

**A:** 
1. 前往「創作者資料庫」→「設定」
2. 點擊「如何取得」按鈕
3. 查看實戰指南中的教學文章
4. 按照步驟取得金鑰（支援 Gemini 和 OpenAI）

### Q4: 生成的腳本可以商用嗎？

**A:** 可以！所有生成的內容都屬於您，可以自由使用於商業用途。

### Q5: 可以匯出生成的內容嗎？

**A:** 可以！支援以下匯出方式：
- **CSV 檔案**：適合在 Excel 中編輯
- **PDF 檔案**：適合列印或分享
- **Excel 檔案**：在創作者資料庫中可匯出所有資料

### Q6: 訂閱後可以取消嗎？

**A:** 可以！您可以隨時在「創作者資料庫」中取消自動續費。取消後仍可使用至訂閱到期日。

### Q7: Mode2 為什麼暫停服務？

**A:** Mode2（AI 顧問對話模式）目前正在進行功能維護和優化，暫時停止服務。如需使用類似功能，建議使用 Mode1（IP 人設規劃模式）進行深度對話。

### Q8: 支援哪些平台？

**A:** 目前支援：
- Instagram Reels
- YouTube Shorts
- TikTok
- 其他短影音平台（可自訂）

### Q9: 生成的腳本長度可以調整嗎？

**A:** 目前主要支援 30 秒短影音腳本。如需其他長度，可以在對話中向 AI 說明您的需求。

### Q10: 可以同時使用多個模式嗎？

**A:** 可以！您可以根據不同需求選擇不同的模式。所有生成的內容都會統一保存在「創作者資料庫」中。
